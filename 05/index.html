form 태그 교안

1. form 태그란?

form 태그는 사용자로부터 정보를 입력받고 서버로 전송하는 역할을 합니다.

회원가입, 로그인, 검색, 문의하기 등 사용자 입력이 필요한 곳에 사용됩니다.

2. form 태그의 주요 속성

속성 설명 예시
action 폼 데이터를 전송할 서버 주소 action="login.php"
method 데이터 전송 방식 (GET, POST) method="POST"
name 폼의 이름 name="loginForm"
enctype 파일 업로드 시 사용 enctype="multipart/form-data"

3. form 태그의 기본 구조

<form action="서버주소" method="POST">
    <label>이름:</label>
    <input type="text" name="username">
</form>

4. 주요 입력 필드

input 타입 설명 예시
text 일반 텍스트 입력 <input type="text">
password 비밀번호 입력 (숨김) <input type="password">
email 이메일 입력 <input type="email">
number 숫자만 입력 <input type="number">
checkbox 중복 선택 가능 <input type="checkbox">
radio 하나만 선택 가능 <input type="radio">
file 파일 선택 <input type="file">
submit 제출 버튼 <input type="submit">

5. 로그인 폼 예제

예제 코드:

<form action="login.php" method="POST">
  <fieldset>
    <legend>로그인</legend>
    <label for="id">아이디:</label>
    <input type="text" id="id" name="id"><br>
    <label for="pw">비밀번호:</label>
    <input type="password" id="pw" name="pw"><br>
    <input type="submit" value="로그인">
  </fieldset>
</form>

7. GET vs POST 메서드

구분 GET POST
데이터 위치 URL에 표시 요청 본문에 숨김
보안 낮음 높음
데이터 크기 제한적 (약 2KB) 많음
사용 예 검색, 필터링 로그인, 회원가입

8. 유용한 팁

✓ label 태그 사용하기:
<label for="id">아이디</label>
<input type="text" id="id" name="id">

label의 for 속성과 input의 id를 연결하면 사용성이 좋아집니다!
✓ name 속성 필수:
name 속성이 없으면 서버로 전송되지 않습니다.
반드시 각 입력 필드에 고유한 name을 지정하세요!
✓ fieldset과 legend:
<fieldset>과 <legend>를 사용하면 폼이 논리적으로 그룹화되고 접근성이 향상됩니다.
✓ placeholder 사용:
placeholder는 입력 필드의 예시를 표시합니다.
<input type="text" placeholder="예시 텍스트">
