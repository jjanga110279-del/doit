break문 설명

break문은 반복문(for, while, do-while)이나 switch문을 즉시 종료하는 제어문입니다.

【 기본 사용법 】

1. 반복문에서의 break
   - 반복문 실행 중 break를 만나면 즉시 반복을 중단
   - 가장 가까운 반복문만 종료됨

   예시: for (let i = 0; i < 5; i++) {
          if (i === 3) break;
          console.log(i);  // 0, 1, 2 출력 후 종료
        }

2. switch문에서의 break
   - case 블록의 코드를 실행한 후 switch문을 빠져나감
   - break가 없으면 다음 case로 계속 실행됨 (fall-through)

   예시: switch(num) {
          case 1:
            console.log('일');
            break;  // 반드시 필요
          case 2:
            console.log('이');
            break;
        }

【 주요 특징 】
- 가장 가까운 반복문이나 switch문만 종료
- 중첩된 반복문에서 break는 안쪽 반복문만 종료
- 반복문 밖에서 사용하면 오류 발생
- 프로그램 흐름을 명확하게 제어

continue문은 반복문의 현재 반복을 즉시 종료하고 다음 반복으로 넘어가는 제어문입니다.

【 기본 사용법 】

1. 반복문에서의 continue
   - continue를 만나면 그 이후의 코드는 실행하지 않음
   - 다음 반복(다음 루프)로 바로 진행

   예시: for (let i = 0; i < 5; i++) {
          if (i === 2) continue;
          console.log(i);  // 0, 1, 3, 4 출력 (2는 건너뜀)
        }

2. while문에서의 continue
   - 조건식으로 바로 돌아가서 다음 반복 시작

   예시: let i = 0;
        while (i < 5) {
          i++;
          if (i === 3) continue;
          console.log(i);  // 1, 2, 4, 5 출력 (3은 건너뜀)
        }

【 주요 특징 】
- break와 달리 반복문을 완전히 종료하지 않고 현재만 스킵
- 특정 조건의 데이터를 처리하지 않고 넘어갈 때 유용
- 중첩된 반복문에서 continue는 가장 가까운 반복만 영향
- switch문에서는 사용 불가 (반복문에서만 사용 가능)

【 break vs continue 비교 】
break   : 반복문 전체 종료, 루프 빠져나감
continue: 현재 반복만 건너뜀, 다음 반복 계속 진행
