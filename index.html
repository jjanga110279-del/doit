  1. 그리드 컨테이너에서 열과 행 지정하기 ( display: grid; grid-template-columns: 100px 200px 300px;/* 2개를 쓰면 2열, 3개를 쓰면 3열 */ grid-template-rows: 50px 300px;/* rows 를 쓰면 행을 말한다 )
  2. grid-auto-rows 속성 지정하기 ( grid-template-columns: 1fr 1fr 1fr;/* fr은 공간을 말한다(박스의 크기). 부모요서의 기준으로 2배 */ grid-template-columns: 2fr repeat(2, 1fr);/* 여러개의 동시에 적용할때 repeat 적용함 */ grid-template-rows: minmax(50px,auto);)
  3. 행 높이를 자동으로 지정하기 (grid-template-columns: repeat(3, 1fr); grid-template-rows: minmax(100px, auto);)
  4. auto-fit과 auto-fill을 사용해 배치하기 (/* fit 부모기준으로 박스크기 자동으로 맞춰줌 */ grid-template-columns: repeat(auto-fit, minmax(100px, 1fr) ); /* fill은 100px을 유지하고 남는 공간 그대로 둠 본인기준으로 박스크기맞춤 */ grid-template-columns: repeat(auto-fill, minmax(100px, 1fr));)
  5. 그리드 항목 사이에 간격 지정하기 ( gap:20px 100px /* 공간을 지정해줄때는 gap를 사용함-콘텐츠와 콘텐츠 사이 row 값, , column 값 gap 양쪽 끝의 공간을 줄수없다(가장자리), 가장자리의 여백을 줄라면 마진으로 해야함*/)
  6. 그리드 라인을 사용해 배치하기 (예시:각각 박스마다 grid-column: 1/-1; grid-row: 2/4; 식의 함수를 지정해줘야함 /* -1은 마지막이라는 의미 가변적으로 갯수를 모를때 아이템박스를 한줄 전부를 채울때 -1로 지정해준다. */)
